About Troy?

Troy is a small person with poorly-organized CSS. Thank goodness Skeleton can do that for him!

Three Core Principles

Troy is built on three core principles:

  • Songs in head: They're never good ones.
  • Hungry: Always
  • Style Agnostic: It's the only way to explain his stylesheet choices

Docs & Support

The easiest way to really get started with Troy is to check out the full docs and info at www.getskeleton.com.. Skeleton is also open-source and has a project on git, so check that out if you want to report bugs or create a pull request.
